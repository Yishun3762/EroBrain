# EroBrain 语言

一个基于 Brainfuck 的有(瑟)趣(瑟)编程语言，使用中文字符和符号作为指令。

## 指令集

| 原始Brainfuck | EroBrain | 功能 |
|---------------|----------|------|
| `>` | 嗯 | 指针右移 |
| `<` | 哦 | 指针左移 |
| `+` | 啊 | 增加指针指向的字节值 |
| `-` | 齁 | 减少指针指向的字节值 |
| `.` | 去了 | 输出指针指向的字节 |
| `,` | ？ | 输入一个字节 |
| `[` | ！ | 循环开始 |
| `]` | ♡ | 循环结束 |


## 注意事项

*   确保你的代码文件（如 `hello.ero`）是以 `UTF-8` 编码保存的。
*   为了增加一些气氛，建议将文件后缀命名为.ero,虽然这并不会影响什么
*   为了整齐一点，？和！均采用中文全角符号
